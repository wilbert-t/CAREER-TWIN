## Lesson 11 — Nixpacks won't find Python if requirements.txt is not at repo root
**Session:** 2026-04-09 A
Nixpacks auto-detects Python by looking for `requirements.txt` at the repo root. If it's in a subdirectory (e.g. `career-twin/backend/requirements.txt`), Nixpacks uses the bare Ubuntu image with no Python/pip, causing `pip: command not found` (exit code 127).
**Fix:** Add `providers = ["python"]` at the top of `nixpacks.toml` to force the Python provider regardless of file location.
**Apply when:** Any Railway/Nixpacks deploy where requirements.txt is not at the repo root.

## Lesson 10 — Railway switched default builder to Railpack — nixpacks.toml is ignored
**Session:** 2026-04-09 A
Railway's default builder is now Railpack, which does not read `nixpacks.toml`. Deploying without a `railway.toml` causes `Error creating build plan with Railpack`.
**Fix:** Add a `railway.toml` at the repo root with `[build] builder = "NIXPACKS"` to force Nixpacks. Also set `[deploy] startCommand` there as the canonical start command.
**Apply when:** Any Railway deployment that relies on `nixpacks.toml`.

## Lesson 9 — TextShimmer via `motion.create()` unreliable in Tailwind v4
**Session:** 2026-04-08 C
Using `motion.create(Component as any)` with `backgroundPosition` animation and Tailwind arbitrary CSS variable classes (`[--base-color:...]`) does not render reliably in Next.js + Tailwind v4. The shimmer gradient never appeared.
**Fix:** Bypass the abstraction entirely — use a plain `motion.h1` with inline `style={{ backgroundImage: "...", backgroundSize: "200% 100%", WebkitBackgroundClip: "text", WebkitTextFillColor: "transparent" }}` and animate `backgroundPosition` directly. No component wrapper needed.
**Apply when:** Any text shimmer effect in this project. Skip TextShimmer component for actual usage.

## Lesson — Preview_match_score needs Pydantic range validation
**Session:** 2026-04-07 B
A plain `int` field with a comment `# 0–100` is not enforced. Pydantic will accept 150 or -5.
**Fix:** Use `Field(..., ge=0, le=100)` whenever a numeric field has a bounded range.

## Lesson — LLMs often wrap JSON in markdown fences — strip before json.loads
**Session:** 2026-04-07 B
Groq returns ` ```json\n{...}\n``` ` even when instructed not to. `json.loads()` throws `JSONDecodeError`.
**Fix:** Always strip fences: `if content.startswith("```"): content = content.split("\n",1)[-1]; content = content.rsplit("```",1)[0].strip()`. Apply to every LLM service.

## Lesson — "Load failed" on Safari = fetch() network error, not an HTTP error

## Lesson — Shell state does not persist between Bash tool calls
**Session:** 2026-04-07 A

**Fix:** Always use full venv binary paths: `.venv/bin/pytest`, `.venv/bin/pip`, etc.
