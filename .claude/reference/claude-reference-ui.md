# Parallel Futures - UI Reference

## Product framing
This is not just a CV analyzer. It should feel like a personal AI assistant that helps students understand where they are now, where they could go next, and what they should build to stand out. [file:1]

---

## Main screens

### 1. Landing page
- Product tagline
- PDF upload CTA
- Clean demo-first feel [file:1]

### 2. Analysis loading state
- Animated status text such as:
  - reading your profile
  - mapping future paths [file:1]

### 3. Dashboard
- Left or top: current state card
- Center: 3 future path cards
- Right or bottom: selected detail panel [file:1]

### 4. Path detail drawer or modal
- Role description
- Why it fits
- Trajectory
- Skill gaps
- Recommended projects [file:1]

### 5. Project detail drawer
- Stack
- Build plan
- Impact
- Uniqueness
- Relevance [file:1]

### 6. Assistant chat panel
- Similar ideas
- More unique
- Easier
- More real-world [file:1]

### 7. Roadmap section
- Separate 30 / 90 / 180-day section after project selection [file:1]

## Scope control
Must-have:
- PDF upload
- Current state analysis
- 3 career paths
- 3 projects minimum per path
- one project detail view
- roadmap section
- polished UI [file:1]

Nice-to-have but cut first:
- charts
- export
- too many role categories
- too much chat complexity
- multi-language [file:1]
